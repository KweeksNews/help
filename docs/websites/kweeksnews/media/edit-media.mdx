## Menghapus media melalui akun

1. Masuk ke akun elemen/kelompok Anda.
2. Layangkan kursor pada menu **KweeksNews** pada bilah navigasi atas, lalu klik menu **Dasbor**.
3. Klik menu **Media** pada bilah navigasi kiri.
4. Layangkan kursor pada judul kiriman, lalu klik menu **Sunting**.
5. Edit judul, teks alternatif, keterangan, dan deskripsi.
6. Klik tombol **Perbarui** untuk menyimpan perubahan.
